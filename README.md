## How to Run
### Client to Server
- Server
``` 
python3 Server.py [port] [filepath]
```
- Client
``` 
python3 Client.py [port] [server_port] [output_filepath]
```

### Peer to Peer
```
python3 FileNode.py [port]
```

### Tic Tac Toe
```
python3 Tictactoe.py [port_1] [peer_port]
python3 Tictactoe.py [port_2] [peer_port]
```
